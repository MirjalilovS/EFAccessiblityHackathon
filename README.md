EF hack


Idea: An E-commerce service for blind people
Key components

- Automatically identify items though RPI camera (invoked through intent classification) (what type of item, condition, size, description, etc)
- Web scraper (potentially astroid used for web interaction, python for stats and calculations)
- Elevenlabs conversational agent for invoking these services (https://elevenlabs.io/conversational-ai)
- Astroid.ai creates listing on already logged in depop account
